# NAME
mp4-converter - Convert an mp4 video to webp|ogv optimized for web and save the first frame as jpeg

# INSTALLATION
- `sudo chmod +x /path/to/mp4-converter.sh`

# USAGE
- `mp4-converter.sh input-video.mp4`

# REQUIREMENTS
- ffmpeg
- file
